Query URI Example

Note:The example queries below are based on the Test instrument and its detectors. Make sure you create a test instrument before running the Query URI for an instrument URI. Make sure you create test detectors before running the Query URI for the detectors' examples below.

GET /hascoapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestInstrument - Returns the Test instrument's JSON.
GET /hascoapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestSubcontainer1 - Returns the Test Subcontainer1's JSON.
