Testing RESTful API

Test Container (Instrument and Subcontainer)

GET    /hascoapi/test/instrument/create - Create a test instrument.
GET    /hascoapi/test/instrument/delete - Delete a test instrument.
GET    /hascoapi/test/subcontainer/create - Create a test subcontainer. This method requires the test instrument to exist before it is called.
GET    /hascoapi/test/subcontainer/delete - Delete a test subcontainer.
GET    /hascoapi/test/containerslot/create - Create two containerSlots for the test instrument and two container slots for the subcontainer. This method requires both the test instrument and the test subcontainer to exist before it is called.
GET    /hascoapi/test/containerslot/delete - Delete containerSlots of test instrument and of test subcontainer, if any. This method requires the test instrument and the subcontainer to exist and for both containers to have containerSlots.

Test Detector Stem

GET    /hascoapi/test/detectorstem/create - Create test detector stem 1 and 2 for a test instrument.
GET    /hascoapi/test/detectorstem/delete - Delete test detector stem 1 and 2.

Test Detector

Note1: Create test detector stems 1 and 2 before creating test detectors

Note2: Create a test instrument along with its containerSlots before creating test detectors

GET    /hascoapi/test/detector/create - Create test detector 1 and 2 for a test instrument.
GET    /hascoapi/test/detector/delete - Delete test detector 1 and 2.
GET /hascoapi/test/detector/attach - Attach test detectors 1 and 2 to test instrument.
GET /hascoapi/test/detector/detach - Detach test detectors 1 and 2 from test instrument.

Test Codebook

GET    /hascoapi/test/codebook/create - Create test codebook.
GET    /hascoapi/test/codebook/delete - Delete test codebook.
GET    /hascoapi/test/codebookslot/create - Create two codebook slots for the test codebook. This methods requires the test codebook to exist before it is called.
GET    /hascoapi/test/codebookslot/delete - Delete codebook slots of test codebook, if any. This method requires the test codebook to exist and to have codebook slots.

Test Response Option

Note: Create a test codebook before creating test response options

GET    /hascoapi/test/responseoption/create - Create test response options 1 and 2 for a test codebook.
GET    /hascoapi/test/responseoption/delete - Delete test response options 1 and 2.
GET /hascoapi/test/responseoption/attach - Attach test response options 1 and 2 to test codebook.
GET /hascoapi/test/responseoption/detach - Detach test response options 1 and 2 from test codebook.

Test Annotation Stem

GET    /hascoapi/test/annotationstem/create - Create test annotation stems 1 and 2.
GET    /hascoapi/test/annotationstem/delete - Delete test annotation stems 1 and 2.

Test Annotation

Note1: Create test annotation stems before creating test annotations

Note2: Create a test instrument before creating test annotations

GET    /hascoapi/test/annotation/create - Create test annotations 1 and 2.
GET    /hascoapi/test/annotation/delete - Delete test annotations 1 and 2.

Test Semantic Variable

Note: Create the following before creating test semantic variables: test entity, test attributes, and test unit.

GET    /hascoapi/test/semanticvariable/create - Create test semantic variables.
GET    /hascoapi/test/semanticvariable/delete - Delete test semantic variables.

Test Entity

GET    /hascoapi/test/entity/create - Create test entity.
GET    /hascoapi/test/entity/delete - Delete test entity.

Test Attribute

GET    /hascoapi/test/attribute/create - Create test attributes 1 and 2.
GET    /hascoapi/test/attribute/delete - Delete test attributes 1 and 2.

Test Unit

GET    /hascoapi/test/unit/create - Create test unit.
GET    /hascoapi/test/unit/delete - Delete test unit.
