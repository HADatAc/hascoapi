Repository RESTful API

GET    /hascoapi/api/repo - Return repository setting information.
- Set a new internal name for the repository. (label) needs to be URL encoded.
- Set a new title for the repository that appears in SIR API's GUI. (title) needs to be URL encoded.
- Set a new description for the repository that appears in the SIR API's GUI. (description) needs to be URL encoded.
- Set a new abbreviation and URL for the repository's default namespace. (abbreviation) and (url) need to be URL encoded.
- Set a new abbreviation and URL for the repository's list of namespaces. (abbreviation) and (url) need to be URL encoded.
GET    /hascoapi/api/repo/namespace/delete - Delete the repository's default namespace, if any.

GET    /hascoapi/api/repo/table/namespaces - Namespaces table.
GET    /hascoapi/api/repo/table/informants - Informants table.
GET    /hascoapi/api/repo/table/languages - Language table.
GET    /hascoapi/api/repo/table/generationactivities - Generation activities table.
GET    /hascoapi/api/repo/table/instrumentpositions - Page positions table.
GET    /hascoapi/api/repo/table/subcontainerpositions - Subcontainer positions table.
