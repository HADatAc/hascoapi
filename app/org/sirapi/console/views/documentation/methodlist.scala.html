RESTful API

Repository

GET    /sirapi/api/repo - Return repository setting information.

Retrieval of SIR Element

Note:Every single SIR element is the instantiation of a concept, which is registered at SIR under an unique URI. Instances of concepts like instruments, instrument types, and detectors are example of SIR elements.

GET    /sirapi/api/uri/(instrUri) - Return a full description of a concept as described in SIR. URI needs to be URL encoded.

Instruments

GET    /sirapi/api/instrument/all - Return all instruments in SIR, including their properties.
POST /sirapi/api/instrument/create/(instrumentJson) - Create a new instrument according to the json specification provided. Json needs to be URL encoded. Use the Create Instrument Example to see how to use a CURL with POST method to create instruments.
POST /sirapi/api/instrument/delete/(instrUri) - Delete an existing instrument using the URI provided. URI needs to be URL encoded. Use the Delete Instrument Example to see how to use a CURL with POST method to delete instruments.

Instrument Types

Note: Instrument types are loaded into SIR through the VSTOI ontology. The VSTOI ontology needs to be modified and reloaded to create a new instrument type or to delete an existing instrument type.

GET    /sirapi/api/instrumenttype/all - Return all instrument types currently loaded into SIR.

Semantic Support for Instrument Interoperability

GET    /sirapi/api/instrument/tordf/turtle/(instrUri) - Return RDF encoding in Turtle for requested instrument.
GET    /sirapi/api/instrument/tordf/owlxml/(instrUri) - Return RDF encoding in OWL/XML for requested instrument.
GET    /sirapi/api/instrument/tofhir/(instrUri) - Return FHIR encoding for requested instrument.
GET    /sirapi/api/instrument/toredcap/(instrUri) - Return RedCAP encoding for requested instrument.

Semantic Support for Instrument Response Interoperability

GET    /sirapi/api/response/sdd/(instrUri) - Return Semantic Data Dictionary of instrument response for requested instrument.

Semantic Support for Instrument Documentation

GET    /sirapi/api/instrument/totext/pdf/(instrUri) - Return Text encoding in PDF for requested instrument.
GET    /sirapi/api/instrument/totext/plain/(instrUri) - Return Text encoding in Plain for requested instrument.

Test Instrument

GET    /sirapi/api/instrument/test/create - Create a test instrument.
GET    /sirapi/api/instrument/test/delete - Delete a test instrument.
GET    /sirapi/api/instrument/test/tordf/turtle - Return RDF encoding in Turtle for requested instrument.
GET    /sirapi/api/instrument/test/tordf/owlxml - Return RDF encoding in OWL/XML for requested instrument.
GET    /sirapi/api/instrument/test/totext/pdf - Return Text encoding in PDF for requested instrument.
GET    /sirapi/api/instrument/test/totext/plain - Return Text encoding in Plain for requested instrument.
GET    /sirapi/api/instrument/test/tofhir - Return FHIR encoding for requested instrument.
GET    /sirapi/api/instrument/test/toredcap - Return RedCAP encoding for requested instrument.
