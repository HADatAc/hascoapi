Query URI Example

Note:The example queries below are based on the Test instrument and its detectors. Make sure you create a test instrument before running the Query URI for an instrument URI. Make sure you create test detectors before running the Query URI for the detectors' examples below.

GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestInstrument - Returns the Test instrument's JSON.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestInstrument%2FDTS%2F1 - Returns DetectorSlot 1's JSON of Test instrument.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestInstrument%2FDTS%2F2 - Returns DetectorSlot 2's JSON of Test instrument.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestDetectorStem1 - Returns DetectorStem1's JSON of Detector1 of Test instrument.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestDetectorStem2 - Returns DetectorStem2's JSON of Detector2 of Test instrument.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestDetector1 - Returns Detector1's JSON of Test instrument.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestDetector2 - Returns Detector2's JSON of Test instrument.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestCodebook - Returns the Test Codebook's JSON.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestCodebook%2FROS%2F1 - Returns ResponseOption Slot 1's JSON of Test Codebook.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestCodebook%2FROS%2F2 - Returns ResponseOption Slot 2's JSON of Test Codebook.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestResponseOption1 - Returns ResponseOption1's JSON of Test Codebook.
GET /sirapi/api/uri/http%3A%2F%2Fhadatac.org%2Fkb%2Ftest%2FTestResponseOption2 - Returns ResponseOption2's JSON of Test Codebook.
