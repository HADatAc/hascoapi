Repository RESTful API

GET    /sirapi/api/repo - Return repository setting information.
- Set a new internal name for the repository. (name) needs to be URL encoded.
- Set a new title for the repository that appears in SIR API's GUI. (name) needs to be URL encoded.
- Set a new description for the repository that appears in the SIR API's GUI. (description) needs to be URL encoded.
- Set a new abbreviation and URL for the repository's default namespace. (abbreviation) and (url) need to be URL encoded.
GET    /sirapi/api/repo/namespace/delete - Delete the repository's default namespace, if any.

Tables

GET    /sirapi/api/repo/table/informants - Informants table.
GET    /sirapi/api/repo/table/languages - Language table.
GET    /sirapi/api/repo/table/generationactivities - Generation activities table.
